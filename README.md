# Ranne Care – Site Institucional

## Descrição
Site institucional da Ranne Care, empresa de tecnologia voltada 
para o RH monitorar a saúde mental dos colaboradores com apoio de IA.
O site foi desenvolvido como projeto das Agendas 15 e 16 do curso 
Técnico em Desenvolvimento de Sistemas.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Framework W3.CSS
- Font Awesome Icons

## Como abrir o projeto
1. Acesse o repositório no GitHub
2. Clique em Code > Download ZIP
3. Extraia a pasta
4. Dê duplo clique no arquivo index.html
5. O site abrirá no navegador
